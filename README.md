## ğŸ”§ Customization

### Adding Products
Edit the `products` array in `script.js`:
```javascript
{
    id: 31,
    name: "Your Product Name",
    category: "category-name",
    price: 9999,
    originalPrice: 14999,
    discount: 33,
    rating: 4.5,
    reviews: 1000,
    image: "image-url"
}
```

### Changing Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2874f0;
    --secondary-color: #ff9f00;
    --dark-color: #212121;
}
```
